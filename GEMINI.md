# The AI-Native Software Engineering Principles (GEMINI.md)

This document establishes the foundational design mandates for the AI-Native architecture. These principles ensure that software remains reliable, maintainable, and robust, even when logic is generated by AI agents.

## Principle 1: Strict Side-Effect and State Isolation
- **Mandate**: All computational tasks must operate in isolation.
- **Execution**: Data must be written to temporary files or isolated memory buffers. No changes should be applied to the final system state (e.g., the target file) until the task is fully verified.
- **Goal**: Prevent AI-generated hallucinated logic or partial failures from corrupting the production environment.

## Principle 2: Atomic Execution with Partial Rollback (Undo/Redo)
- **Mandate**: Every operation must be treated as an atomic transaction.
- **Execution**: Tasks must implement both `Execute()` and `Undo()`. If an execution fails, the system must automatically trigger a rollback to restore the environment to its pre-task state.
- **Goal**: Enable localized recovery without requiring a full system reset.

## Principle 3: Intelligent Logging and Execution Budgets
- **Mandate**: Resource consumption must be strictly controlled, and diagnostics must be performance-efficient.
- **Execution**: 
  - **Budgets**: Every task must have a defined execution timeout (e.g., 5 seconds) to prevent infinite loops.
  - **Logging**: Use a memory-based Ring Buffer for high-frequency logs. Only dump logs to disk when an "Error Flag" is triggered.
- **Goal**: Maintain high performance while providing "black box" flight recorder capabilities for AI debugging.

## Principle 4: Contract-Driven Design (Sequence Diagram Contracts)
- **Mandate**: Visual and interface contracts take precedence over implementation.
- **Execution**: A sequence diagram must be finalized and committed as the "Contract" before any code is written. The implementation must strictly follow this contract.
- **Goal**: Provide a deterministic blueprint that guides AI generation and ensures architectural alignment.
